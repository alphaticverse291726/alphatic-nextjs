# Alphatic Labs — Next.js

The Operating System for Modern Clinics.

## Getting Started

npm install
npm run dev

Open http://localhost:3000

## Stack

- Next.js 14 (App Router)
- React 18
- Custom CSS (globals.css)

## Structure

src/app/         → pages (file-based routing)
src/components/  → Navbar, Footer, Modal
src/lib/         → ThemeContext, ModalContext
src/data/        → pricing, products, blog
src/styles/      → globals.css
